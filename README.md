# RPG Simulator — Edición Tkinter

Juego de rol por turnos con exploración de mapas y combate táctico,
desarrollado completamente con la biblioteca estándar de Python (Tkinter).

![Versión Python](https://img.shields.io/badge/python-3.10%2B-blue)
![Licencia MIT](https://img.shields.io/badge/license-MIT-green)
![Cero dependencias](https://img.shields.io/badge/dependencies-0-brightgreen)

---

## 🎮 Características

- **Tres clases jugables**: Guerrero (tanque), Mago (hechizos), Arquero (equilibrado).
- **Mapa de 5×5** generado proceduralmente con enemigos aleatorios.
- **Combate por turnos** con ataques normales, habilidades de clase, pociones y huida.
- **Sistema de experiencia y niveles**: los enemigos escalan con tu nivel.
- **Guardado y carga** de partida en archivo JSON.
- **Interfaz nativa** con Tkinter: sin dependencias externas, compatible con Python 3.10+.
- **Pruebas unitarias** para toda la lógica del juego.
